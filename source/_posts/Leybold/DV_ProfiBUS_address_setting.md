步骤如下：

1. 点击“钥匙”图标，进入登录界面。
![](1_key.png)

2. 点击{% button #, Log-in %}按钮。
![](2_login.png)

3. 输入密码“`customing`”，然后点击{% button #, OK %}按钮。
![](3_password.png)

4. 依次点击{% button #, Process %} > {% button #, Communication %}按钮。
![](4_comm.jpg)

5. 在蓝圈内输入ProfiBUS地址，确认红圈内是有打勾的。
![](5_profibus.jpg)

6. PROFIBUS 地址设置完成。
